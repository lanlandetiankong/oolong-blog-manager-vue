## 6、本地部署
1. 先在本地项目文件夹打开cmd
2. cmd输入 git clone https://gitee.com/hashChange/oolong-blog-manager-vue.git
3. npm install 或 cnpm install 或 yarn add
4. 本地运行:  npm run dev

## 7、编译打包
1、 使用Npm对项目进行打包编译(生成的内容在 项目路径/dist 文件夹下)
npm run build
2、将dist拷贝出来放到要部署的文件夹，用nginx指向该文件夹即可。
